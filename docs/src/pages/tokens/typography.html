Typography

Typography carries 80% of the design. The type scale, tracking, weight hierarchy, and monospace labels create the Strand aesthetic without imagery.

Font Stacks

--strand-font-sans

Inter, -apple-system, BlinkMacSystemFont, Segoe UI, system-ui, sans-serif

Primary UI and headline font. Geometric, engineered for screens.

--strand-font-mono

JetBrains Mono, SF Mono, Fira Code, Cascadia Code, monospace

Data, labels, and code display. Excellent legibility at small sizes.

Tracking (Letter-Spacing)

Large text tighter, small text looser. Tight tracking on large text is the single most impactful "premium" adjustment.

Tightest tracking
--strand-tracking-tightest / -0.05em / Display 60px+
Tighter tracking
--strand-tracking-tighter / -0.03em / Large headlines 40-60px
Tight tracking
--strand-tracking-tight / -0.02em / H1-H2, 30-48px
Normal tracking
--strand-tracking-normal / 0 / Body text 16-20px
Wide tracking
--strand-tracking-wide / 0.02em / Monospace data
Wider tracking
--strand-tracking-wider / 0.05em / Small labels
Widest tracking
--strand-tracking-widest / 0.08em / All-caps overlines
Ultra tracking
--strand-tracking-ultra / 0.12em / Spaced-out micro-labels

Weight Hierarchy

Light-weight headlines at large sizes produce thin strokes that feel elegant. The contrast between weight 300 and weight 400 creates hierarchy.

Display headline (300 Light)
Weight 300 + tracking tighter (-0.03em) + leading tight (1.15)
Subheading (500 Medium)
Weight 500 + tracking tight (-0.02em) + leading snug (1.25)
Body text (400 Regular)
Weight 400 + tracking normal (0) + leading normal (1.5)
Overline / Label (500 Medium + Mono)
Weight 500 + uppercase + monospace + tracking widest (0.08em)
Data value (600 SemiBold + Mono)
Weight 600 + monospace + tracking wide (0.02em)
