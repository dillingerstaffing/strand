Elevation

A 5-level shadow system creating z-axis depth. All shadows use blue-shifted rgba values for cohesion with the cool palette.

Elevation Scale

Hover over each card to see how elevation communicates interaction state.

0
--strand-elevation-0
none
Flat, no lift
1
--strand-elevation-1
Cards at rest
2
--strand-elevation-2
Cards on hover, dropdowns
3
--strand-elevation-3
Modals, popovers
4
--strand-elevation-4
Dialogs, toasts

Shadow Values

--strand-elevation-0: none;
--strand-elevation-1: 0 1px 2px rgba(15, 23, 42, 0.04),
                      0 4px 12px rgba(15, 23, 42, 0.02);
--strand-elevation-2: 0 4px 6px rgba(15, 23, 42, 0.04),
                      0 12px 24px rgba(15, 23, 42, 0.06);
--strand-elevation-3: 0 8px 16px rgba(15, 23, 42, 0.06),
                      0 24px 48px rgba(15, 23, 42, 0.08);
--strand-elevation-4: 0 16px 32px rgba(15, 23, 42, 0.08),
                      0 32px 64px rgba(15, 23, 42, 0.12);

Usage

Use elevation transitions to communicate interactivity. Cards rest at level 1 and rise to level 2 on hover.

.card {
  box-shadow: var(--strand-elevation-1);
  transition: box-shadow var(--strand-duration-fast) ease,
              transform var(--strand-duration-fast) var(--strand-ease-out-expo);
}

.card:hover {
  box-shadow: var(--strand-elevation-2);
  transform: translateY(-2px);
}
