Section Rhythm

Consistent vertical rhythm across page sections creates a cadence that feels intentional.

/* Standard section padding */
padding-block: clamp(4rem, 8vw, 8rem);

/* Hero section padding */
padding-block: clamp(6rem, 12vw, 12rem);

/* Section header margin-bottom */
margin-bottom: clamp(2rem, 4vw, 4rem);

/* Overline to heading */
margin-bottom: 0.75rem; /* space-3 */

/* Heading to description */
margin-bottom: 1rem; /* space-4 */

Premium Whitespace

Premium whitespace has four properties:

  • Consistent: All gaps are multiples of 4px.
  • Proportional: Larger elements get more space around them.
  • Directional: More space above a heading than below it.
  • Purposeful: Every gap groups or separates content intentionally.

Test: if you can remove 30% of the whitespace and nothing feels cramped, you had too much.
