Shape

Border radius communicates personality. Strand uses subtle rounding that signals precision without looking corporate (sharp) or playful (round).

Radius Scale

--strand-radius-sm
4px
Tags, badges
--strand-radius-md
6px
Buttons, inputs
--strand-radius-lg
8px
Cards
--strand-radius-xl
12px
Modals, dialogs
--strand-radius-full
9999px
Avatars, pills

Rules

  • Never 0px. Sharp corners feel cold and corporate. Even the smallest elements get --strand-radius-sm.
  • Never 16px or more (except --strand-radius-full for circles). Too much rounding feels playful, which conflicts with Strand's precision aesthetic.

Usage

.tag {
  border-radius: var(--strand-radius-sm);
}

.button {
  border-radius: var(--strand-radius-md);
}

.card {
  border-radius: var(--strand-radius-lg);
}

.modal {
  border-radius: var(--strand-radius-xl);
}

.avatar {
  border-radius: var(--strand-radius-full);
}
