Getting Started

Strand ships design tokens as CSS custom properties and a component library built on those tokens. You can use the tokens alone or with the full component library.

Install

Install the tokens package:

npm install @strand-ui/tokens

Or with pnpm:

pnpm add @strand-ui/tokens

Import the CSS

Strand requires three CSS files loaded in order: tokens (custom properties), reset (cross-browser normalization), and base (global styles including the lab surface).

<link rel="stylesheet" href="@strand-ui/tokens/css/tokens.css">
<link rel="stylesheet" href="@strand-ui/tokens/css/reset.css">
<link rel="stylesheet" href="@strand-ui/tokens/css/base.css">

Or in a CSS file:

@import '@strand-ui/tokens/css/tokens.css';
@import '@strand-ui/tokens/css/reset.css';
@import '@strand-ui/tokens/css/base.css';

Use the Tokens

All tokens are available as CSS custom properties prefixed with --strand-. Use them in your own CSS:

.my-card {
  background: var(--strand-surface-elevated);
  border: 1px solid var(--strand-gray-200);
  border-radius: var(--strand-radius-lg);
  padding: var(--strand-space-6);
  box-shadow: var(--strand-elevation-1);
  font-family: var(--strand-font-sans);
  color: var(--strand-on-surface-elevated);
}

First Component

Here is a Strand Button rendered with pure CSS classes. No JavaScript required for the visual appearance.

<button class="strand-btn strand-btn--primary strand-btn--md" type="button">
  <span class="strand-btn__content">Primary Action</span>
</button>

<button class="strand-btn strand-btn--secondary strand-btn--md" type="button">
  <span class="strand-btn__content">Secondary</span>
</button>
