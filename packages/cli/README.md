# strand-ui

CLI for the [Strand](https://github.com/dillingerstaffing/strand) design language. Auto-detects your framework. Sets up tokens, CSS, and AI agent support.

## Quick Start

```bash
npx strand-ui init
```

Detects Preact/React, Vue, Svelte, or CSS-only. Creates `STRAND.md` (AI agent instruction file), copies design tokens, generates config.

## Commands

```bash
strand init              # Set up Strand in your project
strand add button        # Copy a component into your codebase
strand list              # List all 32 components
```
