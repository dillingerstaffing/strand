# Strand + Vite + Preact Starter

Minimal starter template for building with Strand UI, Vite, and Preact.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173 to see the app.

## AI agent setup

```bash
npx strand-ui init
```

Generates `STRAND.md` with component docs for Cursor, Copilot, and Claude Code.

## Build

`npm run build` then `npm run preview`

Full docs: [github.com/dillingerstaffing/strand](https://github.com/dillingerstaffing/strand)
